# HandTracker 3D Uploader

PHP-приложение для входа по индивидуальному цифровому коду и загрузки 3D-моделей в формате `GLB`.

## Возможности

- вход по цифровому коду без пароля;
- стартовый код администратора: `576-576-576`;
- все коды доступа имеют формат `123-123-123`;
- администратор создает новые коды пользователей вручную или кнопкой генерации;
- у каждого пользователя свой лимит мест для загрузок;
- базовый лимит: `50`, но администратор может увеличить его отдельно для любого участника;
- принимаются только файлы `.glb`;
- для каждого файла создается полная ссылка, которую можно копировать и открывать напрямую;
- доступен предпросмотр `GLB` прямо на основной странице, сразу под загрузчиком;
- файл можно скачать или удалить из списка;
- администратор тоже может загружать свои файлы через тот же простой экран;
- каждый файл хранится `30` дней, затем автоматически удаляется;
- SQLite создается автоматически при первом запуске.

## Что изменено

- интерфейс переведен на Bootstrap 5;
- для пользователей добавлено поле `upload_limit` в базе;
- администратор может задавать лимит мест при создании пользователя;
- администратор может позже менять лимит мест для каждого участника.
- для загрузок добавлен `share_token`, чтобы формировать постоянные полные ссылки;
- предпросмотр модели работает через `model-viewer`.
- интерфейс упрощен: сверху загрузка, ниже просмотр 3D и список файлов.

## Структура

- `index.php` — основной файл приложения;
- `app/bootstrap.php` — инициализация, база, сессии, служебные функции;
- `data/app.sqlite` — база SQLite;
- `storage/uploads/` — хранилище загруженных файлов.

## Запуск

Нужен PHP `8.1+` с включенным расширением `pdo_sqlite`.

Пример запуска встроенным сервером PHP:

```bash
php -S localhost:8000
```

После этого откройте:

```text
http://localhost:8000
```

## Замечания

- автоудаление просроченных файлов запускается при каждом открытии сайта;
- если база уже существует, новое поле лимита добавится автоматически;
- для крупных моделей при необходимости увеличьте `upload_max_filesize` и `post_max_size` в `php.ini`.
